The file `obs-zoom-to-mouse-fixed.lua` is currently the best version, with no errors. You can use it directly without needing to make any further code modifications.

> [!WARNING]
> However, if an error occurs, you can try modifying the script by changing `obs_sceneitem_set_info` to `obs_sceneitem_set_info2`, then save it and refresh.
